hugo Github blog

Github+hugoでブログ開設する流れ

別記事で解説したGithub+hugoでブログ開設する流れはこんな感じ。

  1. Githubのアカウントを取得
  2. PCにgitをインストール今回はここ
  3. PCにエディタをインストール
  4. PCにhugoをインストール
  5. hugoのテーマを選んでPCにインストール
  6. ローカルリポジトリとリモートリポジトリを作成
  7. リモートリポジトリをGithub Pagesに設定
  8. エディタで記事を編集し、hugoを使ってローカルリポジトリで記事作成
  9. 作った記事をリモートリポジトリにpush

Gitインストール方法


今回はgitのインストールを解説してみます。


前提条件


  • OS：Windows10 Home
  • gitバージョン：2.30.1

手順


  1. インストーラをダウンロード

    git for windowsの公式サイトからインストーラをダウンロードする。

    「ダウンロードボタン」をクリックするとダウンロードを開始します。

  2. インストール実行

    ダウンロードしたファイルをダブルクリックして実行します。

    管理者権限が必要です。管理者でログインしていない場合は、右クリックしてメニューから「管理者として実行」を選択。

  3. GNUライセンス同意

    GNUライセンスに同意して、「Next]ボタンを選択。

  4. インストール先設定

    インストール先を入力します。特に変更する必要なければそのままでOK。

    すでにフォルダが存在する場合は、確認画面が表示されるので「はい」で続けます。

  5. オプション選択

    オプションを好みで選択します。特にこだわりなければそのままでOK。

  6. スタートメニュー選択

    スタートメニューに追加するかどうか。普通ならそのままでOK。追加したくなければ、「Don't create a Start Menu folder」にチェックを入れます。

  7. デフォルトエディタの設定

    メッセージ入力が必要となるコマンドを実行すると起動されるエディタ。よく分からなければそのままでOK。

  8. リポジトリを作成した時のブランチ名称

    リポジトリを作成した時のブランチ名を決めれますが、良く分からなければそのまま「Let Git decide」でOK。デフォルトでブランチ名が「master」になります。今のところmasterが主流で、何か分からないことがあった時にググったりしてもmasterで記載されていることが多いため、基本そのままでOK。

  9. PATH環境変数設定

    PATH環境変数に追加するフォルダを選択します。良く分からなければそのままでOK。

  10. SSH接続環境の設定

    SSH接続する時に使うプログラムを設定する。よくわからなければそのままでOK。

  11. HTTPの設定

    サーバ(Github)接続する時の設定。良く分からなければそのままでOK。

  12. 改行コードの自動変換設定

    チェックアウトやコミットの時に、改行コードを自動で変換するかどうかの設定。複数人で作業するのではなく、一人しか使わないのであればそのままでOK。

  13. ターミナルの設定

    Git Bashで使うターミナルの設定。よくわからなければそのままでOK。

  14. git pullの動作設定

    git pullコマンド時のマージの動作を設定する。よく分からなければそのままでOK。

  15. 認証マネージャーの設定

    インストールすると、リモートリポジトリに接続する際に毎回認証するのを省略できます。良くか分からなければそのままでOK。

  16. オプションの設定

    オプションの設定。よく分からなければそのままでOK。

  17. 試験運用オプションの設定

    試験運用中のオプションの設定。何やらできるようになりますが、バグがあるのが分かっているらしいのでチェックなしのままでOK。

  18. インストール開始

    ようやく全部の設定が終わったので、「install」を押してインストール開始。

  19. インストール終了

    「Finish」を押してインストール終了。

    手順6で設定したスタートメニューに何やら新しいメニューが作られているので確認してみて下さい。
    エクスプローラーの何も表示されていないところで右クリックメニューを表示しても、新たなメニューが追加されているので、合わせて確認してみて下さい。


終わりに


無事インストールは完了しましたでしょうか。
