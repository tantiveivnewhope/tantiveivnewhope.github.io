Github+hugoでブログ開設する流れ

別記事で解説したGithub+hugoでブログ開設する流れはこんな感じ。

  1. Githubのアカウントを取得
  2. PCにgitをインストール
  3. PCにエディタをインストール今回はここ
  4. PCにhugoをインストール
  5. hugoのテーマを選んでPCにインストール
  6. ローカルリポジトリとリモートリポジトリを作成
  7. リモートリポジトリをGithub Pagesに設定
  8. エディタで記事を編集し、hugoを使ってローカルリポジトリで記事作成
  9. 作った記事をリモートリポジトリにpush

エディタインストール方法


今回はPCへのエディタのインストールを解説してみます。


前提条件


  • OS：Windows10 Home
  • Visual Studio Code: ver.1.54.2

    • エディタは何かと使い勝手がよく、かつ無料であるVisual Studio Codeを利用します。
    2021年3月13日現在のバージョンは1.54.2となっています。


手順

  1. Visual Studio Code公式サイトにアクセス

    Visual Studio Codeの公式サイトからインストーラをダウンロードする。

    「今すぐダウンロード」をクリックしてダウンロードページに飛びます。

  2. インストーラをダウンロード

    Windowsのダウンロードっぽいエリアをクリックすると普通は自動的にダウンロードが始まります。

    ダウンロードが自動で始まらない場合は、飛んだページで手動ダウンロードをクリックします。

  3. インストール実行

    ダウンロードしたファイルをダブルクリックしてインストール実行し、最初の画面で「同意する」にチェックして「次へ」をクリック。

  4. インストールフォルダの設定

    インストール先のフォルダ名を設定します。
    特にこだわりなければそのままでOK。

  5. スタートメニューの設定

    スタートメニューに登録するフォルダ名を設定します。
    特にこだわりなければそのままでOK。
    スタートメニューに登録しない場合は、「スタートメニューフォルダを作成しない」をチェックします。

  6. その他設定

    デスクトップフォルダへのショートカット作成や、右クリックメニューへの登録などを設定します。
    特にこだわりなければそのままでOK。

  7. インストール開始

    「インストール」ボタンをクリックしてインストール開始。

  8. インストール実行中

    インストールが実行されます。
    しばらく待ちましょう。

  9. インストール完了

    インストールが完了したら、続けてVisual Studio Codeの設定をするので、「Visual Studio Codeを実行する」にチェックが入っているのを確認して、「完了」をクリック。

  10. Visual Studio Code設定(1)日本語設定

    Visual Studio Codeの初期設定では、言語が全て英語になっています。
    日本語化したい方は日本語設定をしましょう。
    英語のままで良い場合は、ステップ10～12はスキップしてOKです。

    Visual Studio Codeが起動したら、Ctrl + Shift + xで拡張機能メニューを表示します。

  11. Visual Studio Code設定(1)日本語設定

    検索窓に「Japanese Language Pack for Visual Studio Code」と入力します。
    検索窓の下にそれらしきモノが表示されるので、「Install」をクリック。

  12. Visual Studio Code設定(1)日本語設定

    インストールが完了すると、右下辺りに再起動を促してくるので、クリックして再起動。

    再起動されると、日本語になっているはず。

  13. Visual Studio Code設定(2)自動保存

    Visual Studio Codeで編集する際に、編集内容を直ちにファイルに反映させたいことが多いです。
    例えば、hugoでブログを書く時にブラウザで内容を見ながら書いている場合、編集している内容がファイルに反映されると、即座にブラウザにもその編集内容が反映されます。
    いちいちファイル編集＝＞ファイル保存という手数を踏むのは面倒ですね。
    ということで、自動保存の設定にしてしまいましょう。

    自動保存の設定が不要な場合はステップ13～16はスキップしてOKです。

    Visual Studio Codeのメニューから、ファイル＝＞ユーザー設定を選択します。

  14. Visual Studio Code設定(2)自動保存

    検索窓に「auto save」と入力すると、自動保存に関する設定がリストで表示されます。

  15. Visual Studio Code設定(2)自動保存

    自動保存のモードを選択します。

    • off・・・自動保存しない設定
    • afterDelay・・・編集された時、次のステップで設定する時間経過したら自動保存
    • onFocusChange・・・編集中のタブがフォーカスを失うと自動保存
    • onWindowChange・・・Visual Studio Code自体がフォーカスを失うと自動保存

  16. Visual Studio Code設定(2)自動保存

    ステップ15でafterDelayを選択した場合は、保存するまでの時間を設定します。
    Auto Save Delayの入力箇所に時間をミリ秒で設定します。
    デフォルトは1000ミリ秒(=1秒)となっています。

終わりに


以上でVisual Studio Codeのインストールと主な設定は完了です。
他にも設定はたくさんあるので、お好みで設定を変更して下さい。
