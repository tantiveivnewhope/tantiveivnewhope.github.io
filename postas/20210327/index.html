hugo Github blog

Github+hugoでブログ開設する流れ

別記事で解説したGithub+hugoでブログ開設する流れはこんな感じ。

  1. Githubのアカウントを取得
  2. PCにgitをインストール
  3. PCにエディタをインストール
  4. PCにhugoをインストール
  5. hugoのテーマを選んでPCにインストール今回はここ
  6. ローカルリポジトリとリモートリポジトリを作成
  7. リモートリポジトリをGithub Pagesに設定
  8. エディタで記事を編集し、hugoを使ってローカルリポジトリで記事作成
  9. 作った記事をリモートリポジトリにpush

hugoテーマインストール方法


今回はPCへのhugoテーマのインストールを解説してみます。


前提条件


  • OS：Windows10 Home

手順

基本的には、

  1. テーマを選ぶ
  2. 選んだテーマをダウンロードしてインストールする
だけです。

では、さっそく行ってみましょう！

  1. テーマを選ぶ

    テーマとは、要するにブログのフォーマットだと思ってください。 トップページやブログ本体のページ構成が定義されています。 すでにたくさんのテーマが存在するので、自分がこんな風に書きたいな、と思ったテーマをダウンロードして使用しましょう。

    公式ページにものすごくたくさんテーマが置いてあるので、お好みのものを探してみて下さい。
    こんな感じでズラっとリスト化されています。

    画面左にはカテゴライズごとにリスト化されていますので、ここから探してみるのも良いでしょう。

    気になったテーマがあれば、テーマ画像をクリックして個々の詳細なページに移動してみましょう。
    テーマにもよりますが、詳細ページにはだいたい「Download」、「Demo」、「Homepege」の3つのボタンがあります。
    「Demo」をクリックすると、そのテーマのトップページだけではなく、ブログ本体やその他ページ、動作など、テーマ全体の内容を確認することができます。

    結構膨大な数があるので、気に入ったものが見つかるまでじっくり探してみて下さい！

  2. ダウンロード/インストールする

    テーマの詳細画面で「Download」ボタンを押してダウンロードページに移動します。

    ほとんどのテーマは「Download」ボタンを押すと、githubのページに移動します。

    一番簡単なインストール方法はコードをダウンロードして解凍する方法です。
    しかしながらテーマにもよりますが、ページの下の方を見ると、Quick Startedだとか、Installというようなお題でインストール方法が記載されている場合があります。
    その場合は、記載されている方法でインストールすることをお勧めします。
    筆者の場合、「Blonde」というテーマを使用していているのですが、最初単純にコードをダウンロードして解凍したところ、静的ページを作成する段階でエラーが出まくり、らちが明かないので素直にインストールし直しました。
    インストール方法はテーマによって違ってくるので、良く読んでみて下さい。

  3. 単純にコードをダウンロードして解凍する場合

    githubのページ上部に「Code」のボタンがあるのでクリック。
    さらに、出てきたメニューから「Download ZIP」を選択すると、コードファイルのダウンロードが開始されます。

    ダウンロードしたファイルを解凍し、前回hugoコマンドで作成したサイトのフォルダ「themes」の下に、解凍したフォルダごとマルっと移動させます。
    移動後のフォルダ構成のイメージはこんな感じです。
    例えば作成したサイトのフォルダが、c:\hugoだとすると、
    c:\hugo\themes\hugo-coder-master\
    「hugo-coder-master」の箇所はテーマによって変わります。
    また、「hugo-coder-master」の箇所は好きなフォルダ名に変更しても良いみたいです。（未検証）

  4. 最後に動作確認

    これでインストール自体は完了です。
    動作確認しておきましょう。

    コマンドプロンプトでhugoのフォルダに移動して、hugoコマンドでサーバを起動させます。
    (上の例では、c:\hugoに移動します)

    hugo server

    すると、サーバが起動し、

    Web Server is available at http://localhost:1313/ (bind address 127.0.0.1)
    みたいに表示されますので、ブラウザでhttp://localhost:1313/にアクセスします。
    うまくインストールされていれば、テーマが準備してくれているサンプル画面が表示されるはずです。

最後に

以上でhugoテーマのインストールは完了です。
次回はリポジトリの設定をしていきます
