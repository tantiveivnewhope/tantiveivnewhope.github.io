hugo Github blog

Github+hugoでブログ開設する流れ

別記事で解説したGithub+hugoでブログ開設する流れはこんな感じ。

  1. Githubのアカウントを取得
  2. PCにgitをインストール
  3. PCにエディタをインストール
  4. PCにhugoをインストール今回はここ
  5. hugoのテーマを選んでPCにインストール
  6. ローカルリポジトリとリモートリポジトリを作成
  7. リモートリポジトリをGithub Pagesに設定
  8. エディタで記事を編集し、hugoを使ってローカルリポジトリで記事作成
  9. 作った記事をリモートリポジトリにpush

エディタインストール方法


今回はPCへのhugoのインストールを解説してみます。


前提条件


  • OS：Windows10 Home

手順

今回は簡単です！
ダウンロードして解凍するだけ。
最長10分！

  1. hugoのgithubにアクセス。

  2. バージョンを選択します。
    こだわりなければ、最新版でOK。
    バージョンを選びたい場合は、画面左上辺りの「Tags」をクリック。

    バージョンリストが表示されるので、欲しいバージョンをクリック。

  3. ページの下の方に「Assets」というリストがあるので、Windows環境に応じたファイル名をクリックするとダウンロードが始まります。
    Windowsの場合は、64ビット：「hugo_X.XX.X_Windows-64bit.zip」か32ビット：「hugo_X.XX.X_Windows-32bit.zip」です。

  4. ダウンロードしたファイルを解凍します。
    ファイルを右クリックして、「全て展開」を選択します。

    解凍先フォルダを指定して、「展開」をクリックします。

    3つのファイルが解凍されます。

    • hugo.exe
    • LICENSE
    • README.md

  5. 好きなところにフォルダを作成します。
    例：C\hugo

  6. 作成したフォルダ内にbinというフォルダを作成し、binフォルダの下に4で解凍した3つのファイルを全て移動させます。

  7. 次にPATHを通しておきましょう。
    PATHが通っていないと、hugoコマンドを入力した際に、hugo.exeがどこにあるか分からなくてエラーになります。

    Windowsのメニューから「コマンドプロンプト」を起動します。
    以下のコマンドを入力します。
    「c:\hugo\bin」の部分は6で作ったフォルダを指定してください。

    set PATH=%PATH%;C:\hugo\bin
  8. 最後に、サイトを作っておきます。

    好きなフォルダで、サイトを作ります。
    コマンドプロンプトで以下のコマンドを入力します。
    cdコマンドで好きなサイトに移動し、hugoコマンドでサイトを作成しています。
    XXXXXには好きな名前を設定してください。

    cd hogehoge
    hugo new site XXXXX

    フォルダの下に、以下の6つのフォルダと1つのファイルができていると思います。

    • archetypes フォルダ
    • content フォルダ
    • data フォルダ
    • layouts フォルダ
    • static フォルダ
    • themes フォルダ
    • config.toml ファイル


最後に

以上でhugoのインストールは完了です。
次回はhugoのテーマをインストールしていきます
